Aim

Recover deleted files and extract metadata from a disk image using Autopsy and Sleuth Kit.

Theory

The Sleuth Kit is a free digital forensic tool kit used for disk image analysis. Autopsy provides a graphical interface to analyze file systems and recover deleted evidence.

Procedure
Step1

Launch autopsy & create a new case.

Step2

Enter the case information.

Step3

Add disk image, enable modules & start analysis.

Step4

Browse the deleted files & analyze them.

Conclusion

The system forensic tool successfully recovered deleted files and enabled detailed image of system activity and investigation.
