Aim

Create a forensic disk image by FTK Imager or guymager and verify its integrity using SHA-256 hashing.

Objective
To perform digital forensic disk imaging using Guymager & create an Expert witness format image.
To generate and verify hash values (MD5, SHA1, SHA-256) to ensure the integrity of the acquired image.
Methodology
Step1

Launching Guymager.

Step2

Selecting the target device.

Step3

Configure image acquisition & hash calculation.

Step4

Start image acquisition process.

Step5

Generated image files and verified the data integrity.

Conclusion

It can be concluded that guymager can successfully create a forensic image by creating exact image file and maintain data integrity.
